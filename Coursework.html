National Park Website

This is a website group project where we were tasked with creating a website that had connecting data in a database. We were tasked with creating a API for the 4 Data sets that we were supposed to have. Our team made a national park website(similar to yelp but for parks).


List of the locations

My job was to have a List of locations that connected directly to Parks, so when a park was created you could assign it a location so on both the location and park page they would display eachother.


This is what would appear when you click on the California link. In this image you can also see a list of the parks that appear underneath. Those links are done by the api not manually.

The last thing I did for this website was I was tasked with creating something unique(it was very open ended.) So I used an openAPI that was already created to be able to find the distance between a park and a given zipcode.


Unity Game Project

This was week 1 of our meeting where we put together a small platfom a killable cubes.

My tasks in this team has been to plan the development of the game and give everyone on the team tasks every week. Aside from managing the team I primarily work on the movement of the player and UI/inventory system of the character.
